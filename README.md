# Simple Gradle Calculator 

A basic Android calculator app built using **Gradle**. This project supports essential arithmetic operations and includes constants like π (Pi).

---

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`×`)
- Division (`÷`)
- Power (`^`)
- Modulus (`%`)
- Equals (`=`)
- Constant Pi (`π`)

---

##  Tech Stack

- **Language**: Java
- **Build Tool**: Gradle
- **Platform**: Android
